TokyoShelf Bulk Shipment Calculator

Connect a Google Sheet to save and sync shipments, products, and shipping rates. Uses Google Apps Script as the backend — no OAuth or server required.

  1. Open your Google Sheet → Extensions → Apps Script
  2. Delete all existing code, paste the full contents of TokyoShelf_AppScript.js
  3. Click Save (Ctrl+S), then Deploy → New deployment
  4. Type: Web app · Execute as: Me · Access: Anyone → Deploy
  5. Authorize when prompted, then copy the Web App URL
  6. Paste the URL below and click Connect
Stored in your browser only (localStorage). Never transmitted elsewhere.
